## Prompt Shorthand Modes

Speech-to-text friendly shortcuts. Say **`[word] mode`** in your prompt to activate.

### Research & Analysis

| Shorthand | Behavior |
|-----------|----------|
| **drill mode** | Read all relevant code, explore thoroughly, consider multiple hypotheses before acting. Never assume — verify everything first. |
| **xray mode** | Map the complete execution/data flow end-to-end. Trace every layer. |
| **rca mode** | Root Cause Analysis. Don't fix symptoms — ask "why" repeatedly until the actual root cause is found. |

### Completion & Validation

| Shorthand | Behavior |
|-----------|----------|
| **seal mode** | Implement + test + fix until it actually works. Run it and prove it. |
| **prove mode** | Define success criteria, write tests, run them, show green results before declaring done. |
| **clamp mode** | Loop fix → test → fix until zero failures. Don't ask — just keep going. |

### Combos

| Shorthand | Behavior |
|-----------|----------|
| **ironclad mode** | drill + seal + prove. Maximum thoroughness at every stage. |
| **full send mode** | Don't over-ask for clarification. Execute autonomously and deliver. |

### Examples

```
fix the login bug, rca mode, clamp mode
add dark mode to settings, ironclad mode
refactor the auth service, drill mode
deploy the feature, full send mode
```
